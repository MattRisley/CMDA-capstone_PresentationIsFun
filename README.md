# Getting Started
1. Download and install [R](https://www.r-project.org/) and [RStudio](https://www.rstudio.com/) on your local computer. 
2. Install the `rmarkdown` package using the following code in RStudio:
    ```r
    install.packages("rmarkdown")
    ```
